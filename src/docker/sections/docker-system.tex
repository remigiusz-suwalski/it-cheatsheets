\section{Manage Docker itself}
\important{Remove all} unused containers, networks, dangling images, and optionally, volumes:
\begin{bashcode}
$ docker system prune

WARNING! This will remove:
        - all stopped containers
        - all networks not used by at least one container
        - all dangling images
        - all build cache
Are you sure you want to continue? [y/N] y
\end{bashcode}

Useful switches:
\begin{compactitem}
	\item \mintinline{bash}{--force}: without confirmation,
	\item \mintinline{bash}{--prune}: removes all unused images,
    \item \mintinline{bash}{--volumes}: prunes volumes too.
\end{compactitem}
